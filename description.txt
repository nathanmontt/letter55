cozy - project 2

slogans:
  - a place you can stay
  - feels like your home/feels like home
  - dreams can be dreamed

project descitption:
  o projeto visa a criação de uma aplicação web na qual a mesma é referente
  a um site de estilização de interiores de casa. O usuário vai poder entrar
  no site, contando com algumas abas que ele vai poder acessar, que no caso
  serão apenas as de CATÁLOGO DE PRODUTOS, no caso, os interiores, e uma página
  404, uma página de ERRO. Caso seja necessário adicionar mais coisas, serão usadas
  como features futuros para que, assim, o site sempre tenha atualizações quase que
  constantes.

visual idea:
  A ideia visual vai contar com a logo da marca sendo um gato, bem relaxado, para demonstrar
  sim um local mais aconchegante, mais acolhedor, confortável, mais "cozy".

initial features:
  -> Vamos iniciar com um botão de DARK MODE, algo diferente do que costumo fazer.
